PakTrend Store Sales Analysis & Dashboard
📌 Project Overview

This project analyzes sales performance for PakTrend Store using Excel. The dashboard provides insights into customer behavior, regional performance, monthly sales trends, and top-performing sales channels.

📊 Key Insights

Women customers contribute the highest share of sales.

Adults aged 30–49 years are the top-performing customer segment.

KPK is the highest sales-contributing province.

March recorded the highest sales performance.

Amazon is the top-performing sales channel, followed by Daraz and Temu.

📈 Dashboard Features

Monthly sales trend analysis

Customer demographic breakdown

Regional sales comparison

Sales channel performance analysis

Interactive Excel dashboard with slicers and pivot tables

🛠 Tools Used

Microsoft Excel

Pivot Tables & Pivot Charts

Slicers

Data Cleaning & Analysis Techniques

📌 Conclusion

PakTrend Store can improve sales growth by focusing on women customers, expanding marketing efforts in KPK, optimizing Amazon performance, and running seasonal campaigns during high-demand months like March.
